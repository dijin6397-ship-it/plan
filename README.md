# 生产自动排程系统

一个基于Web的生产自动排程系统，支持检修阶段-SBOP(工序)-工单三级结构管理，自动生成甘特图。

## 功能特点

- **三级结构管理**: 检修阶段 → SBOP(工序) → 工单
- **自动排程**: 输入开始时间和作业时长，自动计算排程
- **甘特图展示**: 直观展示排程结果
- **Excel导入**: 支持Excel表格数据导入
- **每日8小时工作制**: 自动跳过非工作时间和午休时间

## 快速开始

### 方法一：使用启动脚本（推荐）

双击运行 `start.bat` 文件，系统会自动：
1. 检查Python环境
2. 安装必要依赖
3. 启动Web服务器

然后在浏览器中访问: http://localhost:5000

### 方法二：手动启动

1. 安装依赖:
```bash
pip install -r requirements.txt
```

2. 启动服务器:
```bash
python app.py
```

3. 打开浏览器访问: http://localhost:5000

## 使用说明

### 1. 数据录入

**方式一：手动添加**
- 点击"添加检修阶段"创建检修阶段
- 在检修阶段下添加工序(SBOP)
- 在工序下添加工单，填写作业时长

**方式二：Excel导入**
- 点击"下载模板"获取Excel模板
- 按模板格式填写数据
- 点击"导入数据"上传Excel文件

### 2. 生成排程

1. 设置开始作业时间
2. 设置每日工作时长（默认8小时）
3. 点击"生成排程"按钮
4. 查看甘特图结果

### Excel模板格式

| 检修阶段 | SBOP/工序 | 工单名称 | 作业时长(小时) |
|---------|----------|---------|--------------|
| 第一阶段检修 | 工序A | 工单A-1 | 2 |
| 第一阶段检修 | 工序A | 工单A-2 | 3 |
| 第一阶段检修 | 工序B | 工单B-1 | 1.5 |

## 排程规则

- 每日工作时间: 8:00 - 17:00
- 午休时间: 12:00 - 13:00（自动跳过）
- 按检修阶段顺序依次排程
- 同一检修阶段内的工序依次执行
- 同一工序内的工单依次执行

## 技术栈

- **后端**: Python Flask
- **前端**: HTML/CSS/JavaScript
- **图表**: Chart.js
- **数据处理**: Pandas, OpenPyXL
